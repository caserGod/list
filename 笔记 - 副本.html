HEADgit中把每一次提交串成一条线，为主分支master， 10. 查看当前文 git branch前面有*表示当前所在分支 11. 创建分支 git branch 加分支名 12. git checkout 加分支名 切换分支 13. git merge 加分支名称 合并指令，在master分支中使用git merge v1.1 会将v1.1修改的合并到master中，v1.1中不变 在v1中修改2.txt内容并提交， 切换到master分支，查看2.txt是否被修改， 修改master分支中的2.txt并提交 切回v1分支中查看2.txt是否被修改， 14. 删除分支 git branch -d 分支名称 不能在当前分支删除自己 git branch -D 分支名称 强制删除 15. 创建新仓库，repository name中填写仓库名字， create 1.关联本地仓库与远程仓库， git remote add origin https://github.com/caserGod/test.git 将本地仓库的内容推送、上传。提交到远程仓库， git push -u origin master origin是别名，指代后面的地址
